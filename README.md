Project Description: 
The team imported the HINTSData_2020_clean.csv dataset, which contains health information, including smoking status, exercise habits, BMI, alcohol consumption, and perceived healthcare quality. Dataset was loaded and inspected, showing 2,402 observations and 8 variables of interest. By cleaning and subsetting specific variables, we aim to identify patterns that influence an individual's health status through a public population health survey. We developed a custom R function (my_summary_func) to streamline the generation of statistical summaries for continuous health data. This project allows the team to manage, analyze, and visual real-world health data by using introductory level R to build and strengthen stastical programming through GitHub.
